# Cot Dashboard

Developer-facing dashboard for managing entities, relationships, and API configuration.

## Features

- Entity management (CRUD)
- Field builder
- Relationship designer
- API key management
- Generated API documentation
- Real-time preview

## Tech Stack

- Next.js 16 (App Router, RSC, Server Actions)
- Tailwind CSS + shadcn/ui
- Clerk authentication
- Drizzle ORM

## Development

```bash
bun dev
```

Navigate to `http://localhost:3000`
