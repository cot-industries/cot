# Cot Documentation

Developer-first documentation for Cot platform.

## Structure

```
content/
├── quickstart.mdx       # 5-minute start
├── core-concepts/       # Entity, relationships, etc.
├── guides/              # Tutorials
├── api-reference/       # Auto-generated API docs
└── resources/           # Examples, best practices
```

## Development

```bash
bun dev
```

Navigate to `http://localhost:3001`

## Writing Docs

- Keep it concise and practical
- Show code examples first
- Progressive disclosure
- One concept per page
